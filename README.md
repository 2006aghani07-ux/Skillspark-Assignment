# Skillspark-Assignment
Web Technologies Skillspark Assignment

Project Overview
SkillSpark is built using HTML, CSS, Bootstrap Git and Github, and JavaScript to create a responsive landing page for an online learning platform.

The website includes:
1. Responsive navigation bar
2. Hero section with call-to-action
3. Course feature cards
4. Why Choose Us section
5. Student testimonials carousel
6. Dark mode toggle
7. Footer with social links

Technologies Used
1. HTML5
2. CSS3
3. Bootstrap 5
4. Bootstrap Icons
5. JavaScript

Website Sections
1. Navigation Bar
A responsive Bootstrap navbar that includes:
1. Home
2. Courses dropdown
3. About
4. Contact
5. Dark Mode Toggle

2. Hero Section
The hero section introduces the platform with a motivational headline:
"Learn. Build. Launch."
Includes a Get Started button linking to the courses page.

3. Features Section
Highlights the main learning categories offered by SkillSpark:
1. Web Development
2. App Development
3. AI & Data Science

Each feature is displayed using Bootstrap cards with images and descriptions.

4. Why Choose Us
Shows the key benefits of the platform:
1. Expert Mentors
2. Practical Projects
3.Career Guidance
4.Certification
Icons are used for visual representation.

5. Testimonials
A Bootstrap carousel displaying feedback from students who used the platform.

6. Footer Contains:
1. Copyright information
2. Social media icons (Facebook, LinkedIn, GitHub)

Dark Mode Feature
The website includes a Dark Mode Toggle button in the navbar.
